User input:
1. my_path (define in WebClass.py)
   directory of folder containing cache files
2. EmailList (define in WebScraper.py)
   a string containing email addresses of recipients
   E.g. EmailList = 'someone@mako.com,person1@mako.com,person2@mako.com'
3. Frequency (define in WebScraper.py)
   how often the scraper is run
   E.g. Frequency = 60 ie runs every minute
4. IP&port (define in SendEmail.py)
5. Directory.csv
   directory with column1=company, column2=url, column3=page, column4=keywords(lower case, comma separated)
   Example keywords: keyword1,keyword2,keyword3
